## Part 1: PA085/PA086 fetcher 修復 (commit `54ece52`)

### PA085 Understat 修復
- **原因特定**: understat.com の HTML から `var teamsData = JSON.parse(...)` 形式の JS 変数が完全消失 (curl 18KB / Playwright 172KB いずれも 0 件)。現状はテーブルが DOM にクライアントサイドで描画される方式
- **対応**: fetch_understat.py を **Playwright レンダリング後の `<table>` 直接 HTML パース**方式に書き直し
- **結果**: 5リーグ **96 teams 取得成功** (EPL 20 / La Liga 20 / Bundesliga 18 / Serie A 20 / Ligue 1 18)
- stats_feed_reader API 変更不要 (team_slug フィールド追加のみ)

### PA086 FanGraphs 修復
- **原因特定**: FanGraphs は Cloudflare challenge (`Just a moment...`) が Playwright headless でも突破不可。ESPN も AWS WAF challenge で同じ
- **対応**: データソースを **公式 MLB StatsAPI (`statsapi.mlb.com`)** に完全切替 (無認証・Cloudflare なし)
- **計算**: FIP = (13×HR + 3×(BB+HBP) - 2×K) / IP + cFIP / wRC+ proxy = 100 × team_OPS / league_avg_OPS
- **結果**: **30 teams 取得成功** (lgOPS=0.708 / lgERA=4.13 / cFIP=3.13)
- 計算由来値は `wRC_plus_is_proxy: true` / `FIP_is_computed: true` フラグで明示
