### 5. トラブルシューティング履歴

| 問題 | 原因 | 解決策 |
|------|------|--------|
| PowerShell `>` エラー | 文字列内の`>`がリダイレクト演算子と解釈 | `→`に変更または削除 |
| ヒアストリング不認識 | Write toolがBOM無しUTF-8で保存、PSはShift-JIS読み込み | `$p = "..." + "..."`の文字列連結に変更 |
| スクリプト文字化け | 同上のエンコーディング問題 | 全スクリプトを英語のみに統一 |
| Start-Processでプロンプト未渡し | 引数の扱いが異なる | `&`演算子に変更 |
| Claudeが2時間無応答 | 対話型プロンプト待ちでハング | `--permission-mode bypassPermissions`追加 |
| gitコミット失敗 | Author identity未設定 | `git config --global user.name/email`設定 |
| GitHub認証拒否 | パスワード認証廃止 | PAT（Personal Access Token）使用 |
